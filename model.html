Module model

This module defines the data model for the simulation. Specifically, it provides classes for representing patches of the simulation grid and a class for representing cells that inhabit these patches.

class Cell (patch: CellPatch, resistance: int, parent: Optional[Cell] = None)

Represents a cell in the simulation.

Parameters

patch : Patch
 
The patch that will contain this cell (added automatically). The patch must be free.
resistance : int
 
The level of resistance to a patch toxicity. It must be between 0 and 9, included.
parent : Optional[Cell]
 

The cell that generated this cell by division or None if this is a first generation Cell. Defaults to None.
